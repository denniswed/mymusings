# Welcome to My Musings

This is your first blog post! This blog is a simple place to share your thoughts and musings.

## How to Use This Blog

1. Write your posts in Markdown format in the `posts/` directory
2. Run `python build.py` to generate the static site
3. Your site will be in the `output/` directory

## Features

- Simple Markdown-based posts
- Clean, readable design
- Optional commenting system support
- Easy to deploy (GitHub Pages, Netlify, etc.)

## Creating New Posts

To create a new post, simply create a new `.md` file in the `posts/` directory with frontmatter like this:

```markdown
---
title: "Your Post Title"
date: 2024-01-15
author: Your Name
tags: [tag1, tag2]
---

Your content here...
```
